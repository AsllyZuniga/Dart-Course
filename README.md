# 🎯 Dart Course

> A hands-on course covering the fundamentals and advanced topics of the Dart programming language.

This repository was created for **learning purposes**. It explores Dart from the ground up — variables, operators, control flow, functions, types, and classes — with practical exercises designed to build efficient and well-structured applications.

---

## 📁 Project Structure

```
Dart-Course/
├── ejercicios/       # Hands-on exercises for each topic
└── .vscode/          # Editor configuration
```

---

## 🗂️ Topics Covered

| Topic | Description |
|-------|-------------|
| **Variables** | Declaration, types, and null safety |
| **Operators** | Arithmetic, logical, and relational operators |
| **Control Flow** | `if`, `else`, `switch`, loops |
| **Functions** | Named, anonymous, arrow functions |
| **Types** | Lists, Maps, Sets, and generics |
| **Classes** | OOP, constructors, inheritance, and advanced patterns |

---

## 🚀 Getting Started

1. Install [Dart SDK](https://dart.dev/get-dart)
2. Clone the repository
   ```bash
   git clone https://github.com/AsllyZuniga/Dart-Course.git
   ```
3. Run any exercise
   ```bash
   dart run ejercicios/filename.dart
   ```
